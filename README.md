# CodingBook
Tutorial site for the assignment

Files added:

- index.html — main tutorial page
- css/styles.css — styling
- js/script.js — small interactivity (nav + quiz)

Run locally:

1) Open index.html directly in a browser.

2) Or serve with Python:

cd /workspaces/CodingBook
python3 -m http.server 8000
Then open http://localhost:8000 in your browser.

Customize lesson/assignment content in index.html. Ask me to scaffold a build pipeline if you want one.
