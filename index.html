Overview

This tutorial walks you through the assignment step-by-step with short lessons, code samples, and a small quiz to check understanding.

Lessons

Lesson 1 — Problem Breakdown

Read the assignment carefully, identify inputs/outputs, and break the problem into small functions.

Lesson 2 — Implementation Plan

Create a simple plan, write pseudocode, then implement and test incrementally.

Lesson 3 — Testing & Edge Cases

Write tests for normal and edge inputs. Consider invalid input handling and performance.

Assignment

Implement the required functionality described by your instructor. Use the lessons above as a guide. A typical deliverable contains:

  • Source code with clear function boundaries
  • README describing how to run and test
  • Small set of tests or example inputs/outputs

Starter Checklist

  1. Read the problem statement.
  2. Write pseudocode.
  3. Implement core functions.
  4. Run simple tests.
  5. Refactor and document.
