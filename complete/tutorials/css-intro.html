    ការណែនាំអំពី CSS

    បន្ទាប់ពីរៀន មេរៀននេះរួចហើយប្អូននឹងអាច
    ▷ ស្វែងយល់អំពី CSS
    ▷ ស្គាល់ syntax នៃ CSS
    ▷ ប្រើ CSS ដើម្បីផ្លាស់ប្តូររូបរាងវេបសាយ

    អ្វីទៅជា CSS?

    ▷ CSS មកពីពាក្យ Cascading Style Sheets ប្រើសម្រាប់ផ្តល់រូបរាង (style) ដល់ HTML

    ▷ CSS អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងពណ៌ font ទំហំ និង layout នៃវេបសាយ

    ▷ CSS makes web pages beautiful and responsive.

    CSS Syntax

    selector {
    property: value;
    property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 32px;
}
