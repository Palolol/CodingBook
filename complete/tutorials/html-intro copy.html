    ការណែនាំអំពី HMTL

    បន្ទាប់ពីរៀន មេរៀននេះរួចហើយប្អូននឹងអាច
    ▷ បង្កើតនូវវេបសាយដែលមានលក្ខណៈ Static
    ▷ ចេះឌីហ្សាញវេបសាយ អោយត្រូវតាមស្តង់ដានៃការងារ
    ▷ ស្វែងយល់អំពីការបង្កើតវេបសាយបែប Responsive
    ▷ Deploy វេបសាយជាលក្ខណៈ Global

    អ្វីទៅជា HTML?

    HTML បានបង្កើតឡើងក្នុងកំឡុងទស្សវត្សន៍ទី ៩០ ដែលមាន 3 versions ផ្សេងគ្នា

    ▷ HTML មកពីពាក្យ Hyper Text Markup Language ដែលប្រើសម្រាប់កំនត់នូវ content និង structure នៃ Web Page មួយ

    ▷ HTML ជាភាសាគោលដែលត្រូវសិក្សាតំបូងគេបង្អស់មុន ពេលឈានដល់ការបង្កើតនូវវេបសាយជាមួយនឹង ភាសាផ្សេងទៀត (PHP, Java, Python)

    ▷ HTML has a defined set of tags you can use. You can't make up your own tags to create new styles or features.

    HTML Tags

    📖 Core Concept

    HTML uses tags wrapped in angle brackets to structure web pages. Every tag like <h1> or <p> tells the browser how to display the content inside it.

    Most tags come in pairs — an opening tag and a closing tag (with a / before the name). The content lives between them.
