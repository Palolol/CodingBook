    CSS Selectors

    Selectors tell CSS which elements should receive a style.

    Element selectors target tags.Class selectors use a dot: .cardID selectors use a hash: #header

    What you should know

    Specificity
    ID selectors are stronger than class selectors.
    Tip
    Prefer classes for reusable styles.
    Practice
    Style a card using a class selector and a hover state.

    Quick example

    .card {
  border: 1px solid #333;
}
#header {
  color: #3ea6ff;
}

    Tip: use the sidebar to move through the learning path.

    Key points

    • Element selectors target tags.
    • Class selectors use a dot: .card
    • ID selectors use a hash: #header
