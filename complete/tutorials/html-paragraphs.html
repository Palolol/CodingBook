    HTML Paragraphs

    បន្ទាប់ពីរៀន មេរៀននេះរួចហើយប្អូននឹងអាច
    ▷ ប្រើប្រាស់ <p> tag ដើម្បីបង្កើត paragraph
    ▷ យល់ដឹងពីភាពខុសគ្នារវាង <p> និង <br>
    ▷ រៀបចំអត្ថបទ អោយមានភាពងាយអាន
    ▷ ប្រើប្រាស់ text formatting tags គ្រប់ប្រភេទ

    អ្វីទៅជា HTML Paragraphs?

    HTML Paragraph គឺជា block of text ដែលត្រូវវេទិការ browser បង្ហាញជាផ្នែកអត្ថបទដាច់ដោយឡែកពីគ្នា ដោយមានចន្លោះ (margin) ខាងលើ និងខាងក្រោម។

    ▷ <p> tag ប្រើសម្រាប់រុំអត្ថបទ ហើយ browser នឹងបន្ថែម margin ដោយស្វ័យប្រវត្តិ ដើម្បីញែក paragraph ចេញពីគ្នា។

    ▷ <br> tag គឺជា self-closing tag ប្រើសម្រាប់បំបែកបន្ទាត់ ដោយមិនបង្កើត paragraph ថ្មី។ គួរប្រើតែក្នុងករណីចាំបាច់ប៉ុណ្ណោះ។

    ▷ Browsers automatically add space before and after each paragraph. You cannot control this with blank lines in your HTML — you need CSS for spacing.

    ឧទាហរណ៍ <p> tag

    <p>This is the first paragraph of text.</p>
<p>This is the second paragraph of text.</p>
<p>Each paragraph is automatically spaced apart.</p>

    ភាពខុសគ្នា <p> vs <br>

    <!-- Using paragraph tags -->
<p>First paragraph.</p>
<p>Second paragraph.</p>

<!-- Using line break -->
<p>
  First line.<br>
  Second line in the same paragraph.
</p>

    Text Formatting Tags

    <p>This is <strong>bold text</strong>.</p>
<p>This is <em>italic text</em>.</p>
<p>This is <mark>highlighted text</mark>.</p>
<p>This is <small>smaller text</small>.</p>
<p>This is <del>strikethrough text</del>.</p>
<p>This is <ins>underlined text</ins>.</p>

    ច្បាប់ល្អៗ (Best Practices)

    ▷ ប្រើ <p> tags ដើម្បីរុំ paragraph នីមួយៗ — កុំប្រើ <br> ច្រើនដូចជា paragraph separator។

    ▷ រក្សាអត្ថបទនៃ paragraph ឲ្យខ្លី និងពាក់ព័ន្ធ ដើម្បីងាយអានបំផុត។

    ▷ ប្រើ CSS property ដូចជា line-height និង margin ដើម្បីគ្រប់គ្រង spacing ជំនួសការប្រើ <br> ច្រើន។

    Step 4 / 7
    HTML Paragraphs

    📖 Core Concept

    The <p> tag defines a paragraph in HTML. Browsers automatically add a margin above and below each paragraph to create visual separation between blocks of text.

    The <br> tag inserts a single line break inside a paragraph without starting a new one. Unlike <p>, it is a self-closing tag — it has no closing tag.
