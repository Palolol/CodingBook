    What is CSS?

    CSS មកពីពាក្យ Cascading Style Sheets ប្រើសម្រាប់ផ្តល់រូបរាង (style) ដល់ HTML elements។

    CSS អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រង ពណ៌, font, ទំហំ និង layout នៃវេបសាយ។

    📖 Core Concept

    CSS uses selectors to target HTML elements and apply styles to them.

    Each rule has a property and a value separated by a colon, ending with a semicolon ;

    CSS 
    /* CSS Rule Structure */
selector {
    property: value;
}

/* Real Example */
h1 {
    color: #2965f1;
    font-size: 32px;
    font-weight: bold;
}
    Visual Anatomy
    Understanding the CSS rule structure
    h1 {
    color : #2965f1 ;
    font-size : 32px ;
    }
