    What is CSS?

    CSS មកពីពាក្យ Cascading Style Sheets ដែលប្រើសម្រាប់ផ្តល់រូបរាង (style) ដល់ HTML elements។


    CSS អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រង ពណ៌, font, ទំហំ, និង layout នៃវេបសាយ។

    🎨
    គ្រប់គ្រងពណ៌ និង typography
    📐
    រៀបចំ layout ដោយ Flexbox / Grid
    📱
    បង្កើត responsive design
    ✨
    បន្ថែម animations & transitions
    💡
    Did you know? CSS was first proposed by Håkon Wium Lie in 1994 and released in 1996. Without CSS, all websites would look like plain text documents!

    The building blocks of CSS

    📖 Core Concept

    CSS uses selectors to target HTML elements and apply styles to them.

    Each rule has a property and a value separated by a colon : and ending with a semicolon ;

    CSS 
    /* CSS Rule Structure */
selector {
    property: value;
    property: value;
}

/* Real Example */
h1 {
    color: #2965f1;
    font-size: 32px;
    font-weight: bold;
}
