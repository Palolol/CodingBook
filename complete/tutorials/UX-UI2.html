UX/UI Design Questions & Answers

English Questions (1-10)

1) What is UX Design?
User Experience (UX) Design is the process of creating products that provide meaningful, intuitive, and relevant experiences to users. It involves the entire journey of acquiring, integrating, and interacting with the product.
2) UX design mainly focuses on:
The overall feel of the experience, user satisfaction, solving user problems, and the logical flow of how a user interacts with a product.
3) UI design mainly focuses on:
The visual and interactive elements of a digital product. This includes layout, typography, colors, buttons, icons, and the overall aesthetic look.
4) Which of the following is part of UX design work?
User research, creating personas, journey mapping, wireframing, prototyping, and usability testing.
5) What is the purpose of wireframes?
To act as a low-fidelity, structural blueprint of a webpage or app. They focus on layout, content placement, and functionality without getting distracted by visual design details like colors or fonts.
6) What is a prototype?
An interactive, preliminary model of a product used to test ideas, user flows, and functionality before committing to final development.
7) Which UX process step comes first?
Empathize (or User Research). Understanding the user's needs, behaviors, and pain points is always the starting point.
8) Which UX process step involves interviews and surveys?
User Research (often occurring during the "Empathize" phase).
9) What does usability measure?
How easy, efficient, and intuitive it is for a user to navigate an interface and accomplish their specific goals.
10) What principle ensures a product works well for people with disabilities?
Accessibility (often abbreviated as a11y).

Khmer Questions (11-20)

11) តើ UX Design គឺជាអ្វី?
UX (User Experience) Design គឺជាដំណើរការនៃការបង្កើតផលិតផលដែលផ្ដល់នូវបទពិសោធន៍ល្អ មានអត្ថន័យ និងងាយស្រួលប្រើប្រាស់សម្រាប់អ្នកប្រើប្រាស់ចុងក្រោយ។
12) តើ UX និង UI ខុសគ្នាយ៉ាងដូចម្ដេច?
UX ផ្ដោតលើបទពិសោធន៍ទូទៅ ភាពងាយស្រួល និងដំណើរការនៃការប្រើប្រាស់ (How it works and feels)។ ចំណែក UI ផ្ដោតលើរូបរាង ការរចនា និងសោភ័ណភាព (How it looks) ដូចជាការរៀបចំពណ៌ អក្សរ និងប៊ូតុង។
13) តើ UX Designer មានភារកិច្ចអ្វីខ្លះ?
មានតួនាទីធ្វើការស្រាវជ្រាវ (User Research) ស្វែងយល់ពីបញ្ហាអ្នកប្រើប្រាស់ គូរ Wireframes និង Prototypes និងធ្វើតេស្តសាកល្បង (Usability Testing)។
14) តើ Wireframe គឺជាអ្វី?
ជាគំនូសប្លង់ព្រាង (Blueprint) ដំបូងនៃគេហទំព័រ ឬកម្មវិធី ដែលបង្ហាញពីការរៀបចំទម្រង់ និងទីតាំងទិន្នន័យ ដោយមិនទាន់បញ្ចូលពណ៌ ឬរូបភាពលម្អិត។
15) តើ UX Design Process មានអ្វីខ្លះ?
ជាទូទៅដំណើរការនេះរួមមាន៖ Empathize (ស្វែងយល់ពីអ្នកប្រើប្រាស់), Define (កំណត់បញ្ហា), Ideate (ស្វែងរកគំនិត), Prototype (បង្កើតគំរូសាកល្បង), និង Test (ធ្វើតេស្ត)។
16) តើ User Research គឺជាអ្វី?
គឺជាការស្រាវជ្រាវដើម្បីប្រមូលទិន្នន័យ និងស្វែងយល់ពីតម្រូវការ អាកប្បកិរិយា និងការលំបាករបស់អ្នកប្រើប្រាស់ តាមរយៈការសម្ភាសន៍ ឬស្ទង់មតិ។
17) តើ Information Architecture គឺជាអ្វី?
ជាការរៀបចំ រចនាសម្ព័ន្ធ និងបែងចែកប្រភេទព័ត៌មាននៅក្នុងកម្មវិធី ឬគេហទំព័រ ដើម្បីឱ្យអ្នកប្រើប្រាស់ងាយស្រួលស្វែងរកអ្វីដែលពួកគេចង់បានដោយមិនវង្វេង។
18) តើ Usability មានន័យដូចម្ដេច?
គឺជារង្វាស់ដែលបញ្ជាក់ថាប្រព័ន្ធ ឬកម្មវិធីមួយមានភាពងាយស្រួលក្នុងការប្រើប្រាស់ និងមានប្រសិទ្ធភាពកម្រិតណា។
19) តើ Consistency ក្នុង UX Design មានន័យដូចម្ដេច?
ភាពស៊ីសង្វាក់គ្នា គឺការរក្សាទម្រង់ ពណ៌ ពុម្ពអក្សរ និងមុខងារឱ្យដូចៗគ្នាទូទាំងកម្មវិធី ដើម្បីកុំឱ្យអ្នកប្រើប្រាស់មានការភាន់ច្រឡំ និងងាយស្រួលចងចាំ។
20) ហេតុអ្វី Accessibility មានសារៈសំខាន់ក្នុង UX?
ព្រោះវាធានាថាផលិតផល ឬកម្មវិធីអាចប្រើប្រាស់បានដោយមនុស្សគ្រប់គ្នា រួមទាំងអ្នកដែលមានពិការភាពផងដែរ (ឧទាហរណ៍៖ អ្នកមានបញ្ហាគំហើញ ឬការស្តាប់ជាដើម)។
